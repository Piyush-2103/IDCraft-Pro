                           IDCRAFT PRO
               Enterprise-Grade Bulk ID Card Generation Suite

OVERVIEW
--------------------------------------------------------------------------------
IDCraft Pro is a high-performance desktop application engineered to 
streamline high-volume ID card production. Featuring a sleek, modern interface, 
it bridges the gap between complex database management and print-ready visual 
design. 

Whether you are printing single-sided visitor badges or dual-sided corporate IDs, 
IdentityForge handles the heavy lifting of data binding, image masking, and print 
grid compilation, allowing institutions, event organizers, and businesses to 
generate thousands of custom ID cards in minutes.


CORE FEATURES
--------------------------------------------------------------------------------
* Dynamic Data Binding: Instantly map Excel (.xlsx/.xls) database columns to 
  visual text and automated QR codes using simple {ColumnName} syntax.
* Dual-Sided Support (FULL): Design and generate both the front and back of 
  your ID cards simultaneously.
* Visual Drag-and-Drop Workspace: A rich, interactive canvas to place Text, 
  Images, and QR Codes exactly where you need them.
* Advanced Media Handling: 
  - Auto-Serial image assignment (perfect for numbered photo directories).
  - Bind dynamic images based on database cell values.
  - Apply custom shape masks (Circles, Ovals, Custom PNG masks) & dynamic borders.
* Live Data Preview: Toggle a live preview to see exactly how your raw Excel 
  data will look on the rendered card before exporting.
* Smart Grid Compiler: Automatically compile individual cards into print-ready 
  8x12 or A4 grids. Features custom column/row gaps, automatic cut-lines, 
  card mirroring, and 90-degree rotation.
* High-Quality Exports: Export in JPEG, PNG (Lossless), PDF (Standard Document), 
  or scalable Vector SVG formats.
* Project Management: Save your entire workspace setup as a .bip (Bulk Identity 
  Project) file and load it later.


GETTING STARTED & WORKFLOW
--------------------------------------------------------------------------------
1. Starting a New Project
   - Launch IDCraft Pro and click on "Launch ID Card Software".
   - Select "New project".
   - Map your assets: 
     * Select your Excel Data file.
     * Select the folder containing all your portrait photos/images.
     * Select your background templates (Front and Back).
   - Set your desired canvas dimensions (mm, cm, in, or px) and print DPI 
     (default is 300).

2. The Workspace Editor
   - Adding Nodes: Use the sidebar to add Text, Image, or Data Code (QR) nodes 
     to the canvas.
   - Mapping Data: Select a text node. In the properties panel, enter text like 
     "Name: {Student_Name}" to dynamically pull from the "Student_Name" column.
   - Formatting: Customize fonts, typography (bold/italic), colors, alignment, 
     and bounding box widths. (Text auto-wraps if it exceeds the box width).
   - Focus Mode: Double-click on either the Front or Back canvas to enter a 
     distraction-free focus mode.

3. Output Configuration
   - Click "Continue to Output" once your design is ready.
   - Configure your print grid if you want the software to arrange the cards 
     on a printable sheet.
   - Select your desired export format.
   - Click "Process Batch Generation" to render all ID cards.


DOWNLOAD & INSTALLATION
--------------------------------------------------------------------------------
IDCraft Pro is available as a standalone Windows installer.
1. Navigate to the GitHub Releases Page 
   (https://github.com/Piyush-2103/IDCraft-Pro/releases/latest).
2. Download the latest "IdentityForge-Setup-v1.0.0.exe" file.
3. Run the installer and follow the on-screen prompts.
4. Launch the software and enter your License Key to activate the FULL version, 
   or proceed in Trial Mode.


LICENSING & VERSIONS
--------------------------------------------------------------------------------
IDCraft Pro operates on a robust licensing model to provide a seamless 
trial experience while protecting premium features.

[ Trial Version ]
Upon installation, a 7-Day Trial is automatically activated.
- Limitations: Limited to generating 20 cards per day.
- Restricted Features: Dual-Sided Generation, Data Codes (QR), High-Res PNG/PDF 
  exports, and Vector SVGs are disabled.

[ FULL Version ]
Unlock the ultimate potential of the software by purchasing a license key.
- Unlimited daily generations.
- Unlocks Dual-Sided printing modes.
- Unlocks dynamic QR Data Codes.
- Unlocks High-Resolution PNG, compiled PDF sheets, and Vector SVG outputs.
- Unlocks Grid Rotation and Mirroring options.

*To upgrade, click "Buy FULL Version" on the Welcome screen, complete the UPI 
payment, and enter your 16-character license key.*
