 🪪 IdentityForge Studio

IdentityForge Studio is a professional, enterprise-grade bulk ID card generation suite. Designed for seamless database binding, custom grid processing, and rapid high-resolution print generation, it allows institutions, event organizers, and businesses to generate thousands of custom ID cards in minutes.

---

 ✨ Core Features

 Bulk Generation via Excel: Seamlessly bind `.xlsx` or `.xls` database files to your ID card templates.
 Dual-Sided Support (FULL): Design and generate both the front and back of your ID cards simultaneously.
 Visual Drag-and-Drop Workspace: A rich, interactive canvas to place Text, Images, and QR Codes exactly where you need them.
 Dynamic Data Binding: Use simple `{ColumnName}` syntax to automatically pull data from your Excel sheet into text fields and QR codes.
 Advanced Media Handling:  Bind dynamic images based on database cell values.
     Auto-Serial image assignment (perfect for numbered photo directories).
     Apply custom shape masks (Circles, Ovals, Custom PNG masks) and dynamic borders.
 Live Data Preview: Toggle a live preview to see exactly how your raw Excel data will look on the rendered card before exporting.
 Smart Grid Compiler: Automatically compile individual cards into print-ready 8x12 or A4 grids. Features include:
     Custom column and row gaps.
     Automatic column and row separator lines (cut-lines).
     Card mirroring and 90° rotation.
 High-Quality Exports: Export in JPEG, PNG (Lossless), PDF (Standard Document), or Vector SVG formats.
 Project Management: Save your entire workspace setup as a `.bip` (Bulk Identity Project) file and load it later.

 🚀 Getting Started & Workflow

 1. Starting a New Project
 Launch IdentityForge Studio and click on Launch ID Card Software.
 Select New project.
 Map your assets:  Select your Excel Data file.
     Select the folder containing all your portrait photos/images.
     Select your background templates (Front and Back).
 Set your desired canvas dimensions (mm, cm, in, or px) and print DPI (default is 300).

 2. The Workspace Editor
 Adding Nodes: Use the sidebar to add Text, Image, or Data Code (QR) nodes to the canvas.
 Mapping Data: Select a text node. In the properties panel, enter text like `Name: {Student_Name}` to dynamically pull from the "Student_Name" column in your Excel file.
 Formatting: Customize fonts, typography (bold/italic), colors, alignment, and bounding box widths. Note: The software automatically wraps text that exceeds the defined box width.
 Focus Mode: Double-click on either the Front or Back canvas to enter a distraction-free focus mode.

 3. Output Configuration
 Click Continue to Output ➔ once your design is ready.
 Configure your print grid if you want the software to arrange the cards on a printable sheet.
 Select your desired export format.
 Click Process Batch Generation to render all ID cards.

 🔑 Licensing & Versions

IdentityForge Studio operates on a robust licensing model to provide a seamless trial experience while protecting premium features.

 Trial Version
Upon installation, a 7-Day Trial is automatically activated. 
 Limitations: Limited to generating 20 cards per day.
 Restricted Features: Dual-Sided Generation, Data Codes (QR), High-Res PNG/PDF exports, and Vector SVGs are disabled. 

 FULL Version
Unlock the ultimate potential of the software by purchasing a license key.
 Unlimited daily generations.
 Unlocks Dual-Sided printing modes.
 Unlocks dynamic QR Data Codes.
 Unlocks High-Resolution PNG, compiled PDF sheets, and Vector SVG outputs.
 Unlocks Grid Rotation and Mirroring options.

To upgrade, click "Buy FULL Version" on the Welcome screen, complete the UPI payment, and enter your 16-character license key.
